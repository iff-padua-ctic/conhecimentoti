### Ferramenta simples para cadastro de conhecimento adquiridos durante a manutenção de equipamentos e suporte ao usuário

<p> Para acessar a ferramenta: executar docker-compose build app e docker-compose up -d </p>
<p> acessar http://localhost:8000/login/admin para acessar a manutenção de usuários  </p>
<p> login: admin; senha: admin </p>
<p> Na manutenção de usuários adicionar um novo editor  </p>
<p> acessar http://localhost:8000/login/editor para poder cadastrar novos guias/tutoriais  </p>
